# React E-commerce Website

> A modern, fully responsive **E-commerce web application** built using **React 18+, React Router DOM, Bootstrap 5, Swiper, and Sass**.

> This project includes complete shopping functionality like product listing, filtering, cart, checkout, authentication, wishlist, and user account management.

---

## Features

* React 18 Functional Components & Hooks
* React Router DOM Page Navigation
* Bootstrap 5 UI + Responsive Layout
* Product Grid & List View
* Search, Category & Price Filters
* Wishlist Functionality
* Cart & Checkout Flow
* Authentication (Login / Register / Forgot Password)
* My Account (Orders, Address, Payments)
* Blog Pages
* Fully Responsive (Mobile + Tablet + Desktop)
* Swiper Sliders (Hero, Products, Brands)
* Clean & Scalable Folder Structure

---

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/react-ecommerce.git
cd react-ecommerce
npm install
```

Install required packages:

```bash
npm install react-router-dom@latest
npm install bootstrap@5.2.3
npm install swiper
npm install sass   # optional
```

Run the project:

```bash
npm start
```
